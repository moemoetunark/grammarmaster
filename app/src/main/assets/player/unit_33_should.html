လုပ္သင့္သလား…လုပ္ရင္ေကာင္းမလား လို႕ေမးခ်င္တဲ့အခါ Should  + S + V1? ျဖင့္ေျပာပါ.

·         Should I learn English? ကၽြန္ေနာ္အဂၤလိပ္စာ သင္သင့္သလား

·         Should I speak English? ကၽြန္ေနာ္အဂၤလိပ္စကားေျပာသင့္သလား

·         Should I come to you? မင္းဆီကို ငါလာသင့္သလား

Section B

အသံဖိုင္ပါ ဝါက်မ်ား

·         Tom should go to be earlier. တြမ္ေစာေစာအိပ္ရာဝင္သင့္တယ္. He goes to bed very late and he’s always tired. သူကအျမဲတမ္းအိပ္ရာဝင္ေနာက္က်ျပီး အျမဲေမာပန္းေနတယ္

·         It’s a good film. အဲ့ဒါက ရုပ္ရွင္ကားေကာင္းတကားပဲ. You should go and see it. မင္းသြားၾကည့္သင့္တယ္

·         When you play tennis, you should always watch the ball. တင္းနစ္ကစားတဲ့အခါ မင္းေဘာလံုးကိုအျမဲေစာင့္ၾကည့္သင့္တယ္

Section C

·         Tom shouldn’t go to bed late. တြမ္အိပ္ရာဝင္တာ ေနာက္မက်သင့္ဘူး

·         You watch TV all the time. မင္းတစ္ခ်ိန္လံုး TV ၾကည့္တယ္. You shouldn’t watch TV so much. မင္းတီဗြီအရမ္းမၾကည့္သင့္ဘူး

Section D

I think …………should……. သင့္တယ္လို႕ငါထင္တယ္…

·         I think Carol should buy some new clothes. ကယ္ရို အဝတ္စားသစ္တစ္ခ်ို႕ဝယ္သင့္တယ္လို႕ငါထင္တယ္.

·         It is late. ေနာက္က်ေနျပီ. I think I should go home now. အခုငါအိမ္သို႕သြားသင့္ျပီလို႕ထင္တယ္.

·         A: Shall I buy this coat? ဒီကုတ္အဂ်ၤီကို ငါဝယ္လိုက္ရမလား

B: I think you should. မင္းဝယ္သင့္တယ္လို႕ငါထင္တယ္.

I don’t think ………should……သင့္တယ္လို႕ငါမထင္ဘူး..

·         I don’t think you should work so hard. မင္းအရမ္းပင္ပင္ပန္းပန္းအလုပ္လုပ္သင့္တယ္လို႕ငါမထင္ဘူး

·         I don’t think we should go yet. သြင့္သင့္ျပီလို႕ငါမထင္ဘူး. It’s too early. အရမ္းေစာေသးတယ္

Do you think……………should……..သင့္တယ္လို႕မင္းထင္သလား

·         Do you think I should buy this hat? ဒီဦးထုတ္ကို ငါဝင္သင့္တယ္လို႕မင္းထင္လား

·         What time do you think we should go home? ဘယ္အခ်ိန္အိမ္သို႕ ငါတို႕သြားသင့္တယ္လို႕မင္းထင္သလဲ

Section E

Must သည္ should ထက္ ပိုျပီးျပင္းထန္ပါတသည္

·         It’s a good film. ဒါက ရုပ္ရွင္ကားေကာင္းတစ္ခုပဲ. You should go and see it. မင္းသြားၾကည့္သင့္တယ္

·         It’s a good film. ဒါကရုပ္ရွင္ကားေကာင္းတစ္ခုပဲ. You must go and see it. မင္းသြားၾကည့္ရမယ္

Section F

Should အစား ought to + V1 ျဖင့္လည္းေျပာႏိုင္ပါတယ္..

·         It’s a good film. ဒါကရုပ္ရွင္ကားေကာင္းတစ္ခုပဲ. You ought to go and see it. မင္းသြားၾကည့္သင့္တယ္

·         I think Carol ought to buy some new clothes. ကယ္ရို အဝတ္စားသစ္တစ္ခ်ို႕ဝယ္သင့္တယ္လို႕ ငါထင္တယ္
